# Vue 3 webpack boilerplate

This is preview version Vue 3, DO NOT USE IN PRODUCTION!

## Includes

- Vue 3.0.0-beta.14
- Vue Router 4.0.0-alpha.12
- Vuex 4.0.0-beta.2
- Typescript 3.9.5
- Babel 7.10.2
- Webpack 4.43.0


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
